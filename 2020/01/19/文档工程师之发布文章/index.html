文档工程师之发布文章

1. 创建账号

文档工程师需要用到github，请先创建github。

2. 安装Git 和 authoring tool

下载并安装以下软件和插件
Git：https://git-scm.com/
VScode：https://code.visualstudio.com/
authoring tool：Docs Authoring Pack

3. Fork 想要发布到的repo

到gitbub上找到相关的docs repo，点击Fork。

4. 将fork到的repo 克隆到本地

在本地新建项目文件夹并在文件夹下克隆fork到的repo到本地

5. 设置本地git remote

主要添加Origin&Upstream的地址，相关命令如下。

6. 创建或修改文章

  • 创建或修改文章 (在此之前，请 git pull upstream master 最新文档)

  • git add 以及git commit

  • git push origin master

7. Pull Request （Origin to Upstream ）

在你的GitHub上fork的repo项目，点击pull request

8. 发布文章

  • 验证请求 PRMerger & Acrolinx & Build validation

  • 同意发布

  • 发布文章
